# flutter_boiler

A boilerplate project created in flutter using GetX.

## Getting Started

This boilerplate have basic implimentation for early stage of flutter application. Splash screen, login, routing, config are scaffold for user to use and customize.

## Boilerplate Features:

- Splash
- Login
- Routing
- Config
- Theme
- GetX

## Folder Structure:

Here is the core folder structure which flutter provides.

```sh
flutter-app/
|- android
|- build
|- ios
|- lib
|- test
```
Here is the folder structure we have been using in this project

```sh
lib/
|- data/
|- helpers/
|- models/
|- services/
|- views/
|- widgets/
|- config.dart
|- locator.dart
|- main_prod.dart
|- main_stag.dart
|- main.dart
|- routes.dart
```
